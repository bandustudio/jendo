# [Jendo](https://jendo.overlemon.com)

**Jendo**

#### Install

`cd` to project's dir and run `npm install` 

#### Serve

To pre-compile & hot-reload for development run `npm run serve`

#### Build

Production-ready with minifed bundle `npm run build`
